# YouTube Comment Hide

> A Chrome extension that hides YouTube comments for a distraction-free viewing experience.

[![Chrome Web Store](https://img.shields.io/chrome-web-store/v/ekooomlnaghfabpbbcihikpakihlhaag)](https://chromewebstore.google.com/detail/youtube-comment-hide/ekooomlnaghfabpbbcihikpakihlhaag)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Features

- Toggle comment visibility with a single click on the extension icon
- Remembers your preference across browser sessions
- Syncs state across multiple tabs and windows

## Install

[**Chrome Web Store**](https://chromewebstore.google.com/detail/youtube-comment-hide/ekooomlnaghfabpbbcihikpakihlhaag)
